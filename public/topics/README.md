# Austin AI Club Topics

Markdown is the archive of record for club topics.

The frontend reads curated session data from `src/data.js`, but the durable version of each club session should always exist here too.

## Archive

- [2026-03-18](./2026-03-18.md) - March club topic list

## Monthly workflow

1. Copy `TEMPLATE.md` to `YYYY-MM-DD.md`
2. Add the tracks, stories, and source links
3. Update `src/data.js` so the frontend matches the Markdown
4. Add presenter `notes` only if they help in slideshow mode
5. Keep Markdown and UI data in sync

## Standard categories

Use the same five categories each session unless there is a very strong reason not to:

1. `Local Builds & Projects`
2. `Agent Infrastructure`
3. `Models & Research`
4. `Security`
5. `Big Tech Moves`

These buckets are meant to be stable across weeks so new stories have an obvious home.
